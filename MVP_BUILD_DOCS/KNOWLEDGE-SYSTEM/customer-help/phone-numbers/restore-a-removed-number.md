# Restore a removed number

## Summary

If you removed a business number, you may be able to **restore** it — but only
while it is still scheduled for removal and permanent removal has not yet
completed. A restored number returns to your account in its prior state. If
restore is no longer available, contact support.

## Applies to

Clinic owners who removed a business number and want it back.

## What this means

After you remove a number, there is a window before it is **permanently removed**.
During that window the number is scheduled for removal but not yet gone:

- **Restore is available only before permanent removal completes.** While the
  number is still in this scheduled state, you can usually restore it.
- **Once permanent removal is complete, restore is no longer possible.** At that
  point the number is released and cannot be brought back; you would need to add a
  new number instead.

We do not promise a fixed restore window (such as a set number of days), so if you
want a number back, restore it as soon as you can.

## What you can do

- Open the **Phone number** section of your account and look for the removed
  number while it is still scheduled.
- If a restore option is available, choose it to bring the number back.
- If you do not see a restore option, the number may already be permanently
  removed — contact support to discuss your options.

## What to expect

- **A restored number returns in its prior state.** It resumes normal use subject
  to its previous approval and billing status — restoring does not reset or
  re-do approval, and it does not change what the number was before.
- **Restoring does not turn texting on by itself.** If the number's texting was
  not active before (for example, it was still waiting for SMS approval),
  restoring it does not make texting active. Texting still depends on the usual
  approval and configuration. See
  [Why SMS is not active immediately](../sms-approval/why-sms-is-not-active-immediately.md).
- **Billing follows next-cycle behavior.** Restoring a number updates your
  next-cycle estimate to include it again. See
  [Current vs next billing cycle](../billing/current-vs-next-cycle.md).

## When to contact support

Contact support if:

- the restore option is not available and you still need the number,
- you are unsure whether a number can still be restored, or
- a restored number is not working as you expected.

Sometimes restoring a number close to its permanent-removal time needs a quick
review on our side. We are happy to help.

Email: **support@missedcallsdental.com**
