## Customer-safe notes

- **Settings only.** v1 records which alerts the owner wants; no email/text is
  delivered yet and no minute counter runs yet. Do not imply notifications are
  being sent.
- **AI answering is not active yet.** Frame the minute alerts as preferences the
  owner can set ahead of the AI answered call feature; never say AI answering is
  live.
- **Cite, don't restate billing values.** The included AI answered call minutes
  come from `config/billing.config.ts` / `BILLING-AND-USAGE-POLICY.md`; the
  90%/100% thresholds come from the policy.
