# UI UX Review

Use this skill to review pages, screens, and flows before release.

Review checklist:
- Purpose clarity: user understands what the page does in seconds.
- Information architecture: content order matches decision flow.
- CTA hierarchy: one primary action is obvious.
- Readability: concise copy and scan-friendly sections.
- Interaction: states are clear for hover, focus, loading, and errors.
- Mobile usability: tap targets, spacing, and responsive layout hold up.

Trust and credibility checks:
- Claims are specific and supportable.
- No fake testimonials or inflated numbers.
- Compliance-sensitive language is professional and conservative.

Dental SMS context:
- Flows should reflect real office behavior: missed call to text reply to follow-up.
- Avoid medical-advice phrasing and spam-like language.

Output format recommendation:
- Blockers (must fix)
- High impact improvements
- Nice-to-have refinements
