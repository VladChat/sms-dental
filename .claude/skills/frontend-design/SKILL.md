# Frontend Design

Use this skill for page and UI design implementation in the Dental SMS project.

Design direction:
- Professional B2B SaaS quality.
- Clear hierarchy and high readability.
- Strong visual trust signals without visual clutter.

Rules:
- Start from a clear user goal for each screen.
- Use consistent spacing, typography, and components.
- Prefer code-based visuals (HTML/CSS/SVG) for infographics.
- Build responsive layouts for mobile and desktop.
- Avoid childish styles, gimmicks, and placeholder visuals.

Dental SMS context:
- Messaging should reflect front desk workflows.
- Show practical value: recover missed callers and accelerate follow-up.
- Reinforce trust and operational reliability in UI content and structure.

Quality checks:
- CTA clarity and visual prominence.
- Contrast and keyboard accessibility.
- Layout stability and predictable interaction states.
