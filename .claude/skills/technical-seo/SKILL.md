# Technical SEO

Use this skill for SEO implementation and audit work.

Core checks:
- Indexability: robots, sitemap, canonical, status codes.
- Metadata: unique title, description, and heading hierarchy.
- Structured data: validate rendered schema, not static assumptions.
- Internal linking: logical paths from core pages.

Performance SEO:
- Track Core Web Vitals (LCP, INP, CLS).
- Optimize images, scripts, and render path.
- Avoid layout shift in key conversion sections.

Content integrity:
- Align page intent with specific keyword clusters.
- Keep claims factual and verifiable.
- Avoid doorway-like thin pages.

Dental SMS context:
- Prioritize pages that explain missed-call SMS workflow and pricing.
- Keep compliance-sensitive terms precise and non-misleading.
